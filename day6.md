# **Data Analysis Training – Daily Diary (Day 6)**

**Date:** ___________

Today was the sixth day of my Data Analysis training. The session focused on **Data Structures**, which are used to organize, store, and manage data efficiently. The topics covered included **Stack, Queue, Linked List, and Tree (Binary Search Tree)**.

The session began with **Stacks**. I learned that a stack is a linear data structure that follows the **Last In, First Out (LIFO)** principle, where the last element inserted is the first one to be removed. I studied the basic stack operations such as **push**, **pop**, **peek**, and **isEmpty**, and understood their practical applications, including function calls, expression evaluation, and undo operations.

Next, I studied **Queues**, which follow the **First In, First Out (FIFO)** principle. I learned about the fundamental queue operations such as **enqueue**, **dequeue**, **front**, and **rear**. The trainer explained how queues are commonly used in scheduling, task management, and printer spooling systems.

The trainer then introduced **Linked Lists**, a dynamic linear data structure where elements, called nodes, are connected using pointers. I learned about the structure of a node, how nodes are linked together, and the advantages of linked lists over arrays for dynamic memory allocation. I also understood basic operations such as insertion, deletion, and traversal of nodes.

Finally, I learned about **Trees**, with a special focus on the **Binary Search Tree (BST)**. I understood that a tree is a hierarchical data structure consisting of a root node, parent nodes, and child nodes. In a Binary Search Tree, each node has at most two children, and the left child contains smaller values while the right child contains larger values than the parent node. I also learned about common operations such as insertion, searching, and traversal of nodes in a BST.

Throughout the session, I practiced implementing these data structures in Python and observed how each structure is suited for different types of problems. I also compared their characteristics, advantages, and applications to understand when each data structure should be used.

Overall, today's training strengthened my understanding of fundamental data structures and their role in efficient data storage and processing. These concepts form an important foundation for problem-solving, algorithm design, and data analysis using Python.
