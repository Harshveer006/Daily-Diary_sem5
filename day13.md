# **Data Analysis Training – Daily Diary (Day 13)**

**Date:** 07-07-2026

Today was the thirteenth day of my Data Analysis training. The session focused on **Matplotlib**, one of the most widely used Python libraries for data visualization. I learned how to create different types of graphs and charts to represent data visually, making it easier to analyze patterns, trends, and relationships.

The session began with an introduction to **Matplotlib** and its `pyplot` module. I learned how to import the library and create simple plots using Python. The trainer explained the importance of data visualization in data analysis and decision-making.

Next, I practiced creating **Line Charts**, which are used to display trends and changes in data over time. I learned how to customize line color, style, width, markers, and labels to improve the appearance of the graph.

The trainer then introduced **Bar Charts** and **Horizontal Bar Charts** for comparing values across different categories. I also learned how to create **Histograms** to visualize the frequency distribution of numerical data and understand how data is spread across different ranges.

After that, I studied **Pie Charts**, which represent data as proportions of a whole. I learned how to display percentages, customize colors, and highlight specific sections using the explode feature.

The session also covered **Scatter Plots**, which are useful for identifying relationships and correlations between two numerical variables. I practiced plotting datasets and interpreting the patterns shown by the points.

In addition, I learned how to customize graphs by adding **titles, axis labels, legends, grid lines, figure size, and colors**. These features help make visualizations more informative, attractive, and easier to understand.

During the practical session, I solved several exercises by creating different types of charts using sample datasets. These activities helped me understand how to choose the appropriate graph for different types of data and how to present information effectively.

Overall, today's session provided a strong foundation in data visualization using Matplotlib. I gained practical experience in creating and customizing various charts and graphs, which are essential skills for exploratory data analysis, reporting, and presenting insights from datasets.
