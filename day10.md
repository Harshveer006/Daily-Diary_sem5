# **Data Analysis Training – Daily Diary (Day 10)**

**Date:** ___________

Today was the tenth day of my Data Analysis training. The session focused on **NumPy**, one of the most important Python libraries used for numerical computing and data analysis. I learned how NumPy provides efficient tools for working with arrays and performing mathematical operations on large datasets.

The session began with an introduction to **NumPy Arrays**. I learned how to import the NumPy library and create one-dimensional and two-dimensional arrays using the `array()` function. The trainer explained the advantages of NumPy arrays over Python lists, such as faster execution, lower memory usage, and support for vectorized operations.

Next, I studied **Array Operations**. I practiced performing arithmetic operations such as addition, subtraction, multiplication, division, and exponentiation directly on arrays. I also learned about element-wise operations and how NumPy automatically performs calculations on all array elements without using explicit loops.

The trainer then explained **Array Slicing and Indexing**. I learned how to access individual elements, rows, columns, and subarrays using indexing and slicing techniques. I also practiced modifying array elements and extracting specific portions of an array for data analysis tasks.

After that, I explored various **Mathematical Functions** available in NumPy. I practiced using functions such as `sum()`, `mean()`, `min()`, `max()`, `std()`, `sqrt()`, and other mathematical operations to analyze and process numerical data efficiently. These functions simplify complex calculations and improve program performance.

During the practical session, I solved several **practice questions** based on array creation, indexing, slicing, mathematical operations, and problem-solving using NumPy. These exercises helped me understand how NumPy can be used to manipulate and analyze data efficiently in real-world scenarios.

Overall, today's session provided a strong foundation in the NumPy library. I learned how to create and manipulate arrays, perform mathematical computations, and apply array operations effectively. These concepts are fundamental for data analysis, scientific computing, and machine learning, and they will be valuable in handling large datasets efficiently.
