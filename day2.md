# **Data Analysis Training – Daily Diary (Day 2)**

**Date:** 23-06-2026

Today was the second day of my Data Analysis training. The session focused on **Python Operators**, which are used to perform different types of operations on variables and values.

The first topic covered was **Arithmetic Operators**. I learned how to perform basic mathematical operations such as addition (`+`), subtraction (`-`), multiplication (`*`), division (`/`), modulus (`%`), floor division (`//`), and exponentiation (`**`). I practiced solving numerical expressions using these operators.

Next, I studied **Bitwise Operators**, which perform operations at the binary (bit) level. I learned about bitwise AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), left shift (`<<`), and right shift (`>>`). These operators are useful in low-level programming and efficient data manipulation.

The trainer then explained **Assignment Operators**, which are used to assign values to variables. In addition to the basic assignment operator (`=`), I learned compound assignment operators such as `+=`, `-=`, `*=`, `/=`, `%=`, `//=`, and `**=` that simplify updating variable values.

After that, I learned about **Comparison Operators**, which compare two values and return either `True` or `False`. The operators covered included equal to (`==`), not equal to (`!=`), greater than (`>`), less than (`<`), greater than or equal to (`>=`), and less than or equal to (`<=`).

The next topic was **Logical Operators**. I understood how the `and`, `or`, and `not` operators are used to combine or modify conditional expressions. These operators are commonly used in decision-making and control flow.

I also studied **Identity Operators**, namely `is` and `is not`, which are used to check whether two variables refer to the same object in memory rather than just having the same value.

Finally, I learned about **Membership Operators**, `in` and `not in`, which are used to check whether a value exists within a sequence such as a string, list, tuple, or dictionary.

Overall, today's session provided a strong understanding of the different types of Python operators and their practical applications. I practiced several examples to understand how these operators work in real programming scenarios, which will be useful for writing efficient Python programs in future data analysis tasks.
