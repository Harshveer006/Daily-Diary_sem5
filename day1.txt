# **Data Analysis Training – Daily Diary (Day 1)**

**Date:** 22/6/26

Today was the first day of my Data Analysis training. I was introduced to the Python programming language, which is one of the most popular languages used in data analysis, data science, and machine learning because of its simple syntax and powerful libraries.

The session began with an **Introduction to Python**, where I learned about Python's features, applications, and why it is widely used in the industry.

Next, I learned about **Python Syntax**, which defines the basic rules for writing Python programs. I understood the importance of proper indentation, correct use of statements, and writing clean, readable code.

After that, I practiced using the **print() statement** to display text, numbers, and the output of expressions on the screen.

The trainer then explained **Variables**, which are used to store data in memory. I learned how to create variables and assign different values to them.

I also studied **Data Types** in Python, including:

* Integer (`int`)
* Float (`float`)
* String (`str`)
* Boolean (`bool`)

The concept of **Type Casting** was introduced, where I learned how to convert one data type into another using functions like `int()`, `float()`, and `str()`.

Next, I learned about the **input() function**, which allows users to enter data during program execution. I practiced taking user input and storing it in variables.

Finally, I learned different methods of **Swapping and Unpacking Variables**. I understood how Python can swap the values of two variables without using a third variable and how multiple variables can be assigned values in a single statement using unpacking.

Overall, the first day of training was very informative. I gained a strong understanding of Python basics, which will serve as the foundation for learning data analysis in the upcoming sessions. I also practiced writing simple Python programs to strengthen my understanding of these concepts.
