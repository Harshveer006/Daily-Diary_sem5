# **Data Analysis Training – Daily Diary (Day 4)**

# **Date:** ___________

Today was the fourth day of my Data Analysis training. The session focused on **User-Defined Functions, Classes & Objects, the `__init__` Method, Single Inheritance, and Multilevel Inheritance** in Python. These concepts introduced the fundamentals of object-oriented programming (OOP), which helps in writing organized, reusable, and efficient code.

The session began with **User-Defined Functions**. I learned how to create my own functions using the `def` keyword. The trainer explained how functions improve code reusability, reduce repetition, and make programs easier to understand and maintain. I also practiced defining functions with parameters, passing arguments, and returning values using the `return` statement.

Next, I studied **Classes and Objects**. I learned that a class is a blueprint for creating objects, while an object is an instance of a class. Classes allow us to group related data and functions together, making programs more structured and modular. I practiced creating classes, defining attributes and methods, and creating objects to access them.

The trainer then explained the **`__init__` Method**, which is a special constructor method in Python. I learned that this method is automatically called whenever an object is created and is mainly used to initialize object attributes with the required values.

After that, I learned about **Single Inheritance**. I understood that in single inheritance, a child class inherits the properties and methods of one parent class. This allows code reuse and helps avoid writing the same code multiple times.

Finally, I studied **Multilevel Inheritance**, where one class inherits from another class, and a third class inherits from the second class. I learned how inheritance works across multiple levels and how derived classes can access the properties and methods of their parent and grandparent classes.

Overall, today's session provided a strong introduction to object-oriented programming in Python. I practiced writing programs using functions, classes, objects, constructors, and inheritance. These concepts will help me write more structured, reusable, and scalable Python programs, which are valuable in data analysis and software development.
