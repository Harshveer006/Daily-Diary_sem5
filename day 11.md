# **Data Analysis Training – Daily Diary (Day 11)**

**Date:** ___________

Today was the eleventh day of my Data Analysis training. The session focused on **Pandas**, a powerful Python library used for data manipulation, analysis, and handling structured datasets. I learned about its core data structures and various techniques for processing and analyzing data efficiently.

The session began with an introduction to **Pandas Series**. I learned that a Series is a one-dimensional labeled array capable of storing different types of data. I practiced creating Series objects, accessing elements, and performing basic operations.

Next, I studied **DataFrames**, the most commonly used data structure in Pandas. I learned that a DataFrame is a two-dimensional table consisting of rows and columns, making it ideal for storing and analyzing structured data. I practiced creating DataFrames from dictionaries, lists, and existing datasets.

The trainer then explained **Indexing and Slicing** in Pandas. I learned how to access specific rows and columns using labels and positions with methods such as `loc[]` and `iloc[]`. I also practiced slicing data to retrieve selected portions of a dataset for analysis.

After that, I learned how to handle **Missing Values** in datasets. I practiced identifying missing data using functions such as `isnull()` and `notnull()`, and learned techniques to remove or replace missing values using methods like `dropna()` and `fillna()`. This is an essential step in data cleaning before analysis.

The session also covered **Grouping Data** using the `groupby()` function. I learned how to group records based on one or more columns and perform aggregate operations such as calculating the sum, mean, count, minimum, and maximum for each group.

Next, I studied **Custom Functions** in Pandas. I learned how to create user-defined functions and apply them to Series or DataFrame columns using methods such as `apply()`. This allows customized data transformations and calculations.

The trainer then explained **Joining DataFrames**, where I learned how to combine data from multiple DataFrames using functions such as `merge()`, `join()`, and `concat()`. These operations are useful when working with related datasets.

Finally, I learned how to **Read CSV Files** using the `read_csv()` function and explored how to inspect datasets using functions like `head()`, `tail()`, `info()`, and `describe()`. I understood that reading and exploring datasets is one of the first steps in any data analysis project.

Overall, today's session provided a comprehensive understanding of the Pandas library. I gained practical experience in creating and manipulating Series and DataFrames, handling missing values, grouping data, applying custom functions, joining datasets, and reading CSV files. These skills are fundamental for data cleaning, transformation, and analysis in real-world data science projects.
