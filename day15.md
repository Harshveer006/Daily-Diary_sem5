# **Data Analysis Training – Daily Diary (Day 15)**

**Date:** 9-07-2026

Today was the fifteenth day of my Data Analysis training. The session focused on **Streamlit**, a Python framework used to build interactive web applications for data analysis and machine learning projects. I learned how Streamlit simplifies the process of converting Python scripts into user-friendly web applications without requiring knowledge of HTML, CSS, or JavaScript.

The session began with an introduction to **Streamlit** and its basic components. I learned how to create a simple web application, display text using headings and markdown, and add interactive widgets such as text inputs, number inputs, buttons, and sliders. The trainer explained how these components help users interact with applications in real time.

During the practical session, I developed a **BMI (Body Mass Index) Generator** using Streamlit. The application accepts a user's height and weight as input, calculates the BMI using the standard formula, and displays the result along with the corresponding BMI category, such as underweight, normal weight, overweight, or obese. This project helped me understand how to collect user input, perform calculations, and display dynamic results in an interactive interface.

I also created a **Bar Graph Generator** using Streamlit. In this application, I learned how to take user-defined data as input and generate bar charts to visualize the information. This activity improved my understanding of integrating data visualization with Streamlit to build interactive dashboards.

Throughout the session, I practiced running Streamlit applications locally, modifying the user interface, and testing the functionality of both projects. These hands-on exercises enhanced my understanding of application development and data visualization.

Overall, today's session provided practical experience in building interactive web applications using Streamlit. Developing the BMI Generator and Bar Graph Generator strengthened my Python programming skills and demonstrated how data analysis applications can be presented through simple, interactive, and user-friendly interfaces. These skills will be valuable for creating dashboards and data-driven applications in future projects.
