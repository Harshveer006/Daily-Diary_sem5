# **Data Analysis Training – Daily Diary (Day 17)**

**Date:** 11 July 2026

Today was the seventeenth day of my Data Analysis training. The session focused on the fundamentals of **Microsoft Power BI Desktop** and the **Power Query Editor**, which are essential tools for importing, cleaning, and transforming data before creating reports and dashboards.

The session began with an introduction to **Power BI Desktop**, where I learned about its interface and the various components used for building reports and visualizations. The trainer explained how Power BI is widely used for business intelligence and data analysis.

Next, I explored the **Power Query Editor**. I learned how to import datasets from different sources and perform data cleaning operations before analysis. The trainer demonstrated various transformation techniques such as removing unnecessary columns, changing data types, renaming columns, filtering rows, and preparing raw data for reporting.

I also learned the process of **loading transformed data into Power BI**, which allows cleaned datasets to be used for creating visual reports and dashboards.

During the practical session, I imported sample datasets, explored the Power Query Editor, applied basic data transformation techniques, and successfully loaded the processed data into Power BI for further analysis.

Overall, today's session provided a strong foundation in Power BI Desktop and Power Query. I gained practical experience in importing, cleaning, transforming, and preparing datasets, which are essential steps before performing data analysis and creating dashboards.
