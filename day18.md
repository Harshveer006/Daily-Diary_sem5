# **Data Analysis Training – Daily Diary (Day 18)**

**Date:** 13 July 2026

Today was the eighteenth day of my Data Analysis training. The session focused on **Power BI Visualizations** and connecting Power BI with different data sources to create interactive and meaningful reports.

The session began with an overview of the various **visualization tools** available in Power BI. I learned how to create charts, graphs, tables, and other visuals to represent data effectively. The trainer explained how different visualizations can be selected based on the type of data and the insights to be presented.

Next, I learned about **Filters and Slicers**, which allow users to interact with reports by selecting specific categories or values. These features make dashboards more dynamic and user-friendly by enabling data exploration.

The trainer also explained the **Report Canvas**, where multiple visualizations can be arranged and organized to create professional dashboards. I learned how to design reports with a clear and structured layout.

Another important topic covered was **connecting Power BI to different data sources**. I practiced importing data from **CSV files**, **XML files**, and **SQL Server databases**. I understood how Power BI can retrieve data from multiple sources for analysis and reporting.

Finally, I learned how to **format visuals** by modifying colors, titles, labels, fonts, legends, and other design elements. These formatting options improve the appearance and readability of reports.

During the practical session, I connected Power BI to different datasets, applied filters and slicers, customized visualizations, and explored various formatting options to create more interactive reports.

Overall, today's session enhanced my understanding of Power BI visualizations and data connectivity. I gained practical experience in importing data from multiple sources and designing interactive reports, which are essential skills for business intelligence and data analysis.
